================================================================================
TROUBLESHOOTING
================================================================================

Issue: Module not found error
Solution: Run CELL 1 first (imports and setup)

Issue: File not found error  
Solution: Update file_path in CELL 2 to point to your Excel file

Issue: Visualizations not showing
Solution: Make sure matplotlib backend is configured:
  %matplotlib inline (add to CELL 1)

Issue: Memory error
Solution: Dataset is large, try:
  - Restart kernel
  - Close other notebooks
  - Use smaller sample of data

Issue: Dates not parsing
Solution: Check date format in your Excel file
  Should be: YYYY-MM-DD format
